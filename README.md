# US Agriculture Production Analysis

## Overview
Designed a relational SQLite database using USDA agricultural production datasets.

## Skills Demonstrated
- Database schema design
- SQL
- Joins
- Subqueries
- Data cleaning
- ERD design

## Database
SQLite

## Tables
- cheese_production
- coffee_production
- egg_production
- honey_production
- milk_production
- yogurt_production
- state_lookup

## Example Questions
- Which states produced over 100 million pounds of cheese in April 2023?
- Which states had no milk production in 2023?
- Compare production across commodities.
